# Portf-lio-Pessoal
Aqui está um projeto completo de um site que representa meu portfólio pessoal
Titulo: Portifolio Pessoal

Descrição:
Este projeto foi criado para apresentar meus trabalhos, estudos e experiencias na area de tecnologia. Aqui eu reuni alguns projetos que fiz durante a faculdade e outros que desenvolvi por conta propria, incluindo sites, prototipos e exercicios praticos.

Objetivo:
O objetivo deste portifolio é mostrar minha evolucao como desenvolvedor e reunir em um unico lugar tudo o que venho aprendendo. Tambem é uma forma de facilitar o acesso de professores, colegas e futuros recrutadores aos meus projetos.

Tecnologias utilizadas
HTML
CSS
Bootstrap
JavaScript

Como funciona
O site esta organizado em partes simples e diretas. Na pagina inicial voce encontra uma apresentacao. Em projetos, voce pode filtrar cada tipo de trabalho pela categoria. Cada card mostra titulo, descricao e imagem do projeto.

Motivacao
Decidi criar este portifolio para ter um espaco onde eu pudesse mostrar minha caminhada na area de Sistemas de Informacao. Quero continuar atualizando o projeto conforme eu evoluir, aprender coisas novas e desenvolver trabalhos mais completos.
